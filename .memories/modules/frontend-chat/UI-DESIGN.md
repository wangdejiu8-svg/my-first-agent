# UI设计实现

## 设计理念

简约现代风格，参考 ChatGPT 的设计语言，注重视觉层级和交互反馈。

## 色彩系统

```css
:root {
  --bg-primary: #f7f7f8;      /* 主背景 */
  --bg-secondary: #ffffff;     /* 次级背景 */
  --bg-tertiary: #ececf1;      /* 三级背景 */
  --accent-primary: #10a37f;   /* 主色调（绿色）*/
  --accent-hover: #0d8c6f;     /* 悬浮状态 */
  --text-primary: #2d333a;     /* 主文字 */
  --text-secondary: #6e6e80;   /* 次级文字 */
  --border-color: rgba(0, 0, 0, 0.1);
}
```

## 视觉层级

### 1. 侧边栏与主区域分离
- 侧边栏：#fafafa（浅灰）
- 主区域：#ffffff（纯白）
- 微妙的色差创造空间感

### 2. 按钮层级
- **主要按钮**：实心绿色背景 + 白色文字（新建对话）
- **次要按钮**：透明背景 + 绿色边框（登录）
- **功能按钮**：灰色线框（切换侧边栏）

### 3. 状态反馈
- **选中状态**：浅绿色背景（rgba(16, 163, 127, 0.1)）
- **悬浮状态**：更浅的绿色背景（rgba(16, 163, 127, 0.05)）
- **焦点状态**：2px 绿色边框

## 交互设计

### 空状态引导
- 欢迎语居中显示
- 4个推荐卡片（2x2网格）
- 卡片悬浮效果：上移2px + 绿色阴影

### 输入框反馈
- 默认：1px 灰色边框
- 聚焦：2px 绿色边框
- 发送按钮：输入为空时自动禁用

### 对话列表
- 每项前有消息图标
- 选中项：绿色图标 + 绿色文字 + 浅绿背景
- 悬浮：浅绿背景

## 响应式细节

- 输入框最大宽度：768px
- 推荐卡片最大宽度：600px
- 侧边栏固定宽度：260px
