# 产品需求文档 (PRD)

## 业务目标

为普通用户提供简洁易用的AI对话应用，支持历史对话管理和个性化设置。

## 用户场景

1. 用户注册/登录后使用AI对话功能
2. 查看和管理历史对话记录
3. 自定义界面主题和个人信息

## 核心功能

### 对话功能
- AI对话交互（Markdown渲染、代码高亮）
- 文件/图片上传
- 复制消息、重新生成、停止生成
- 沉浸对话模式

### 历史管理
- 自动生成对话标题（基于第一句话）
- 支持标题重命名
- 对话列表展示

### 用户系统
- 注册：用户名、邮箱、密码
- 登录：用户名/邮箱 + 密码
- 个人信息管理
- 主题切换（浅色/深色）

## 边界假设

- 仅支持桌面端（最小宽度1024px）
- 不支持时间分组、搜索、删除对话
- 不支持密码强度验证、邮箱验证码
- 未登录用户可访问主页但不能发送消息

## 技术栈

- React + HTML5 + CSS3 + JavaScript
- 后端：Django
