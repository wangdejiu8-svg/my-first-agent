# 记忆文件系统

本目录用于存储项目的核心知识和实现细节，便于快速查阅和复用。

## 目录结构

```
.memories/
├── README.md              # 记忆主目录说明
├── modules/               # 记忆模块目录
│   ├── INDEX.md           # 模块索引
│   └── <业务-主题>/       # 具体模块，如 user-auth
│       ├── README.md      # 模块导航
│       ├── PRD.md         # 产品需求、场景、边界
│       └── FUNCTION-*.md  # 功能实现文档
├── templates/             # 模块模板
└── scripts/               # 速查脚本
```

## 文件说明

| 文件 | 作用 |
|------|------|
| README.md | 模块导航，列出所有文档 |
| PRD.md | 产品需求：业务目标、用户场景、边界假设 |
| FUNCTION-*.md | 功能文档：输入输出、核心逻辑、关键代码、踩坑记录 |

## 命名规范

- 模块目录：`业务-主题`，小写加连字符（如 `user-auth`）
- 功能文档：`FUNCTION-功能名.md`，大写加连字符（如 `FUNCTION-LOGIN.md`）
