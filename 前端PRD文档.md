# AI对话应用 - 前端产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
一个面向普通用户的AI对话应用，提供简洁直观的对话交互体验。

### 1.2 目标用户
普通用户，需要简洁易用的AI对话界面。

### 1.3 技术栈
- React
- HTML5
- CSS3
- JavaScript

---

## 2. 页面结构

### 2.1 页面列表
1. **主页（对话页）** - `/` 或 `/chat`
2. **登录页** - `/login`
3. **注册页** - `/register`
4. **设置页** - `/settings`

### 2.2 页面关系
```
主页 ←→ 登录页 ←→ 注册页
 ↓
设置页（通过用户菜单进入）
```

---

## 3. 功能详细说明

### 3.1 主页（对话页）

#### 3.1.1 整体布局
参考ChatGPT的经典三栏布局：
- **左侧边栏**（约250-300px宽）
- **主对话区域**（自适应宽度）
- **不支持移动端**

#### 3.1.2 左侧边栏

**顶部区域：**
- 新建对话按钮

**中间区域：**
- 历史对话列表（可滚动）
  - 对话标题：自动根据用户第一句话生成
  - 支持点击标题进行重命名
  - ❌ 不支持时间分组
  - ❌ 不支持搜索功能
  - ❌ 不支持删除功能

**底部区域：**
- 未登录状态：显示"登录"按钮
- 已登录状态：显示头像 + 用户名
  - 点击后弹出菜单，包含：
    - 设置
    - 退出登录

#### 3.1.3 主对话区域

**顶部区域：**
- 顶部左侧两个相邻按钮：
  - 沉浸对话模式按钮：点击后隐藏左侧边栏，对话区全屏显示，再次点击恢复
  - 新建对话按钮：点击后创建新的对话窗口

**中间区域：**
- 对话消息展示区（可滚动）
- 支持Markdown渲染：
  - 代码块高亮
  - 加粗、斜体
  - 列表、表格等
- 每条消息支持的操作：
  - 复制消息按钮
  - 重新生成回答（仅AI消息）
- AI回复时显示"停止生成"按钮

**底部区域：**
- 多行文本输入框
  - 支持Shift+Enter换行
  - Enter发送消息
- 上传文件/图片按钮
- 发送按钮

#### 3.1.4 访问权限和交互
- 未登录用户可以访问主页
- 未登录用户尝试输入文字时：
  - 弹出提示："请先登录后使用"
  - 不自动跳转到登录页
  - 用户需手动点击左下角"登录"按钮跳转

---

### 3.2 登录页

#### 3.2.1 页面布局
- 居中显示的登录表单卡片

#### 3.2.2 表单字段
- 用户名/邮箱输入框（支持两种登录方式）
- 密码输入框
- 登录按钮

#### 3.2.3 其他元素
- "还没有账号？去注册"链接 → 跳转到注册页
- 可选：记住我复选框（待定）
- 可选：忘记密码链接（待定）

#### 3.2.4 验证规则
- 基本的非空验证
- 邮箱格式验证（如果输入的是邮箱）

---

### 3.3 注册页

#### 3.3.1 页面布局
- 居中显示的注册表单卡片

#### 3.3.2 表单字段
- 用户名输入框
- 邮箱输入框
- 密码输入框
- 确认密码输入框
- 注册按钮

#### 3.3.3 其他元素
- "已有账号？去登录"链接 → 跳转到登录页

#### 3.3.4 验证规则
- 邮箱格式验证（必须）
- 两次密码一致性验证
- ❌ 不需要密码强度验证
- ❌ 不需要邮箱验证码

---

### 3.4 设置页

#### 3.4.1 访问方式
- 仅通过主页左下角用户头像菜单进入
- 主页顶部不设置入口

#### 3.4.2 功能模块

**个人信息管理：**
- 修改用户名
- 修改头像（上传图片）
- 修改邮箱

**账号安全：**
- 修改密码
  - 需要输入旧密码
  - 输入新密码
  - 确认新密码

**界面设置：**
- 主题切换：浅色/深色模式
- 语言选择（中文/英文等）
- 字体大小调整

#### 3.4.3 不包含的功能
- ❌ AI对话偏好设置（模型选择、参数调整等）

## 4. 交互流程

### 4.1 用户注册流程
1. 用户访问登录页，点击"去注册"
2. 跳转到注册页
3. 填写：用户名、邮箱、密码、确认密码
4. 点击注册按钮
5. 前端验证：邮箱格式、密码一致性
6. 提交到后端
7. 注册成功后跳转到登录页

### 4.2 用户登录流程
1. 用户访问登录页
2. 输入用户名/邮箱和密码
3. 点击登录按钮
4. 验证成功后跳转到主页
5. 主页左下角显示用户头像和用户名

### 4.3 对话流程（已登录）
1. 用户在主页输入框输入消息
2. 可选：上传文件/图片
3. 点击发送或按Enter键
4. 消息显示在对话区
5. AI开始回复（显示"停止生成"按钮）
6. AI回复完成后可以：
   - 复制消息
   - 重新生成回答

### 4.4 对话流程（未登录）
1. 用户访问主页
2. 尝试在输入框输入文字
3. 弹出提示："请先登录后使用"
4. 用户点击左下角"登录"按钮
5. 跳转到登录页

### 4.5 历史对话管理
1. 用户发送第一条消息后，自动创建新对话
2. 对话标题自动根据第一句话生成
3. 用户可点击标题进行重命名
4. 点击历史对话列表中的对话，切换到该对话

### 4.6 设置修改流程
1. 用户点击左下角头像
2. 弹出菜单，选择"设置"
3. 跳转到设置页
4. 修改个人信息/密码/界面设置
5. 保存后返回主页
